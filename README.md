# PHP 動態網站專案

## 📌 專案簡介
本專案為課程期末專題，使用 PHP 與 MySQL 建立一個具備會員系統的動態網站。
功能包含使用者登入、留言系統、公告顯示、討論區以及後台管理與審核機制，呈現完整的前後端整合應用。


## 🧱 系統角色與權限
- **公開使用者**：可瀏覽首頁與公告內容  
- **成員**：登入後可參與討論、留言及查看個人資料  
- **管理員**：可進行公告管理、成員審核與系統維護  

## 💡 主要功能說明

### 📢 公佈欄
- 公告新增、編輯與刪除（需再次確認）
- 交錯式表格顯示
- 分頁機制（一次顯示至多 5 筆資料）
- 浮動方式顯示公告詳細內容

### 💬 討論區
- 支援多分類討論主題
- 可進行單選或多選分類篩選
- 依時間排序顯示
- 不同使用者身分顯示不同操作介面

### 👤 個人資料
- 使用者可修改基本聯絡資訊
- 顯示使用者互動相關統計資訊

### 🔎 審核機制（管理員）
- 成員審核與管理
- 採用與公佈欄一致的表格設計
- 可進行通過或淘汰操作

## 🛠 使用技術
- HTML / CSS
- JavaScript
- PHP
- MySQL
- Apache（課程建置使用 XAMPP）


## 📂 專案結構
```

final/
├─ index.php              # 首頁
├─ db/                    # 資料庫連線與後端處理
├─ private/               # 會員 / 管理者頁面
├─ js/                    # JavaScript 功能
├─ pic/                   # 圖片與多媒體資源
└─ *.css / *.html

```

## ⚙️ 環境需求
- PHP 7.4 以上
- MySQL
- Apache Server
